# Aura-Sync EPC 🧠
### Autonomous Project Command Center
**Silicon Plains 2026 | Smart Manufacturing Track**

> AI-Powered EPC Project Planning & Resource Optimization

## Overview
Aura-Sync EPC is a risk-aware project intelligence dashboard that predicts 
task delays using ML, runs Monte Carlo simulations across 500+ scenarios, 
auto-generates optimized schedules using Google OR-Tools CP-SAT, and accepts 
plain English What-If commands via LLM integration.

## Tech Stack
- **AI Engine:** scikit-learn · Google OR-Tools · NumPy
- **Backend:** FastAPI · Socket.io · PostgreSQL
- **Frontend:** React.js · Plotly.js · D3.js · Tailwind CSS

## Features
- 🔴 Risk-Aware Gantt Chart with Monte Carlo confidence bands
- ⚙️ OR-Tools CP-SAT Schedule Optimizer
- 💬 Natural Language What-If Simulator
- ⚡ Real-Time Collaborative Dashboard
